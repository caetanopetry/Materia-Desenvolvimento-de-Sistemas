Introdução ao Bootstrap

O Bootstrap é um framework front-end que facilita a criação de sites responsivos e modernos. Ele utiliza uma estrutura baseada em HTML, CSS e JavaScript para oferecer componentes prontos e um sistema de grids que se adapta a qualquer tamanho de tela.
